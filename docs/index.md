# Welcome to Commandeck

**Commandeck is a visual command launcher for Linux, macOS, and Windows.** Build a grid of clickable buttons — each one runs a shell command, locally or on a remote server over SSH.

## Who is Commandeck for?

**If you're new to the command line**, Commandeck saves you from memorizing commands. Click a button to check your disk usage, update your system, or reboot — no terminal needed. Commandeck comes with dozens of ready-to-use buttons out of the box.

**If you're a sysadmin or power user**, Commandeck is a visual SSH command launcher. Assign buttons to one or more remote machines, run commands across your infrastructure with a single click, see the output instantly — and drive it all from an AI assistant via the [MCP server](pro/mcp.md).

---

## What makes Commandeck different

- **Install and go — nothing to host.** No server, no Docker, no YAML, no port to expose. Commandeck is a native desktop app: open it and click.
- **SSH built in.** Add a machine once, then pick the target at click time — and one button can run on several machines at once.
- **A visible, organized grid.** Your commands become labeled buttons in categories — no memorizing aliases or remembering which script does what. Edit everything visually in a couple of clicks, no config files to hand-edit.
- **Unlimited buttons, real commands.** As many on-screen buttons as you like, each running full shell or SSH commands — not just keystrokes.
- **Local and private.** Your buttons are plain-text files on your own machine. No cloud, no account, no telemetry.

It works *alongside* your terminal, not instead of it — Commandeck kills the repetitive re-typing; you keep the terminal for interactive work.
